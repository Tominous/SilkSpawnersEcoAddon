This is the README of SilkSpawnersEcoAddon!
This plugin adds an optional economy feature (via Vault) to charge the chaning of spawners
Thanks for using!

This plugin sends usage statistics! If you wish to disable the usage stats, look at plugins/PluginMetrics/config.yml!
This plugin is released under the Creative Commons Attribution-NonCommercial-ShareAlike 3.0 Unported (CC BY-NC-SA 3.0) license.

Standard config:

# You can configure every entityID or a default!
cantAfford: "&e[SilkSpawnersEco] &4Sorry, but you can''t change the mob of this spawner, because you have not enough money!"
afford: "&e[SilkSpawnersEco] &2This action costed &e%money%"
default: 10.5
pig: 7.25
cow: 0.00

Permissions

**General**
silkspawners.free - Bypasses the economy check
